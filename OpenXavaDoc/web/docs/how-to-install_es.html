openxava / documentación / Cómo instalar OpenXava

Primero descarga OpenXava desde openxava.org o desde SourceForge. Escoge la distribución apropiada para tu sistema operativo.
Después descomprime el archivo que has descargado, un zip o un tar.gz, en cualquier carpeta de tu ordenador.
Finalmente, ejecuta openxava.exe (para Windows) u openxava (para Linux y Mac). Puedes encontrar este archivo en la raíz de la carpeta dondes has descomprimido OpenXava.
Para versiones anteriores a las 6.4 no tienes openxava.bat/sh, en su lugar abre quick-start.html con un navegador y sigue sus instrucciones.
