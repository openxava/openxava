openxava / documentation / 32-bit

OpenXava Studio (openxava.exe in Windows or openxava executable in Mac/Linux) requires a 64-bit machine to work. However, you can use OpenXava framework in a 32-bit computer, even the latest version. Just download a 32-bit Java and a 32-bit Eclipse, then switch your Eclipse workspace to use the one included in OpenXava distribution.

Install Java for 32-bit

Go to Oracle JDK 8 download page, there choose a distribution with x86 in the name (or i586 in the filename), like Linux x86 Compressed Archive or Windows x86:
Download it, run it and follow all the steps to install the JDK in your computer.

Download and install OpenXava

Download the latest OpenXava version from openxava.org and uncompress it in your computer. You will get a folder called openxava-6.x with three subfolders: doc, studio and workspace. We're going to use the workspace folder in the next section. There is also an openxava.exe or openxava Linux executable in the root, don't click on it, it does not work with 32-bit.

Download and install Eclipse

Eclipse 2018-09 was the last with 32-bit support, so download Eclipse 2018-09 for Windows 32 or for Linux 32. Uncompress the downloaded file and click on the eclipse executable you have inside the eclipse folder.
