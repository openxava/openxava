openxava / documentation / Java 6 (until v6.0.2)

Since v6.1 OpenXava needs Java 8 or better

By default, OpenXava is compiled with Java 6, so you can use OpenXava with WebSphere 8 and WebLogic 10. However, the application server included in the OpenXava distribution is Tomcat 9 (since OpenXava 6) that requires Java 8. Therefore, you need to use Java 8 (or better) to follow the introduction documentation to OpenXava.
If you want to use Java 6 with OpenXava follow the next steps:
  • Install a JDK 6 in your computer.
  • In Eclipse choose it: Windows > Preferences > Java > Installed JREs : Choose the JDK 6
  • Download and install Tomcat 7: https://tomcat.apache.org/download-70.cgi
  • Configure your Eclipse to use the just downloaded Tomcat 7 instead of the Tomcat included with OpenXava.
