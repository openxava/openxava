openxava / documentación / IntelliJ IDEA con OpenXava

Si estás usando una versión de OpenXava anterior a la 7.0 mira las instrucciones antiguas
OpenXava está basado en Maven por lo que funciona muy bien con IntelliJ.

Crear un proyecto OpenXava nuevo

Crea un nuevo proyecto Maven con IntelliJ, escogiendo org.openxava:openxava-archetype-spanish como arquetipo y seleccionando 7.6 para el campo Version (nunca selecciones 7.X.X para versión):

Fíjate que escogemos Maven Central como repositorio.

Empaqueta y ejecuta tu proyecto

Antes de ejecutar tu proyecto has de hacer un mvn package en él. Después busca la clase tuaplicacion.java (es el nombre de tu aplicación) dentro del paquete org.example.tuaplicacion.run en src/main/java, y ejecutala.

También puedes ver la Lección 1 de primeros pasos usando IntelliJ.
