Change Log for OpenXava 8
=========================

- Java 17 as baseline. Dropped support for Java 8 and 11.
- Jakarta Persistence 3.2 support. Dropped support for JPA 2.2.
- Jakarta Validator 3.1 support. Dropped support for Bean Validation 2.0.
- Hibernate Validator upgraded to 9.1.0.
- Hibernate upgraded to 7.2.15.
